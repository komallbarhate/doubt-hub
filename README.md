DoubtHub is a real-time doubt collection platform designed for classrooms and lectures.
It allows students to submit doubts during lectures while teachers can monitor, analyze, and respond to them instantly.

The system improves classroom interaction, doubt tracking, and lecture clarity.

Students can ask doubts anonymously, while teachers get a live dashboard showing all doubts and confusion.
